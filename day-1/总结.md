# 项目总结

## 项目概述

本项目包含两个示例程序，分别演示 PyTorch 深度学习训练和 CUDA GPU 编程。

## 文件列表

| 文件 | 描述 |
|------|------|
| `1.py` | PyTorch MLP 神经网络训练脚本 |
| `hello_gpu.cu` | CUDA GPU 并行编程示例 |

## 详细说明

### 1.py - PyTorch 神经网络

- **网络结构**：两层全连接网络 (Linear)
  - 输入层: 20 → 隐藏层: 64
  - 隐藏层: 64 → 输出层: 2
- **激活函数**：ReLU
- **优化器**：Adam (lr=0.001)
- **损失函数**：CrossEntropyLoss
- **训练**：10000 个 epoch，批量大小 100
- **功能**：对随机生成的数据进行二分类训练

### hello_gpu.cu - CUDA 编程

- **核函数**：`hello_from_gpu` 在 GPU 上运行
- **并行配置**：2 个 Block，每个 Block 4 个 Thread
- **输出**：打印 "Hello from block X, thread Y!"
- **同步**：使用 `cudaDeviceSynchronize()` 确保完成

## 运行方式

**PyTorch 脚本：**
```bash
python 1.py
```

**CUDA 程序：**
```bash
nvcc hello_gpu.cu -o hello_gpu
./hello_gpu
```

## 技术栈

- Python 3 + PyTorch
- CUDA C/C++
